# iM AIRGen TD
### AI Report Generator for iM Bank Trading Desk

> Presented by **황원철 (WeonCheol Hwang)**
> [Codex](https://openai.com/codex) & [Claude Code](https://claude.ai/code) orchestration으로 제작

---

## 개요

**iM AIRGen TD**는 iM뱅크 자금시장부를 위한 AI 기반 마켓 리포트 자동 생성 시스템입니다.
PoC를 위한 시스템이며 국내외 금융시장 데이터를 자동 수집하고, Anthropic 가 이를 분석·서술하여 임원 및 딜러 배포용 **일간·주간·월간 마켓 브리프**를 즉시 생성합니다. 현재는 한국은행 ECOS, 미 연준 FRED, Yahoo Finance 등 무료 API사용. 따라서 일부 데이터는 시점, 정확도에 문제가 있음. Bloomberg API Token 적용시 즉시 상업적 활용 가능

---

## 주요 기능

| 기능 | 설명 |
|---|---|
| **자금시장 모닝브리프** | 채권·외환·해외 시장을 아우르는 AI 일일 총평 자동 생성 |
| **주간 마켓리뷰** | 주간 금리/환율 변동 요약, 수익률곡선 분석, 차주 전망 |
| **월간 마켓리포트** | 월간 매크로 테마 분석, 크레딧 스프레드, 차월 금리·환율 레인지 |
| **수시 브리핑** | FOMC 결과, 금통위 결정, 시장 급변동 등 on-demand 긴급 분석 |
| **A4 출력 최적화** | 모든 리포트가 A4 비율 미리보기로 렌더링, PDF 내보내기 지원 |
| **모바일/데스크탑 적응형** | 모바일은 탭 방식, 데스크탑은 좌우 분할 레이아웃 |

## ⚠️ 데이터 한계

본 시스템은 **무료 공개 API**를 사용하며, 다음과 같은 제약이 존재합니다:

- **시차** — ECOS 데이터는 전영업일 기준이며, 익일 오전에 반영되는 경우가 있습니다. 장중 실시간 데이터는 제공되지 않습니다.
- **정확도** — Bloomberg Terminal, Reuters Eikon 등 유료 데이터 벤더와 소폭 차이가 있을 수 있습니다.
- **커버리지 한계** — NDF 1M, 스왑포인트, IRS 1Y/5Y, Euribor, 한국 CDS 5Y 등 일부 지표는 무료 API로 제공되지 않아 수동 입력값 또는 기준값으로 대체됩니다.
- **안정성** — Yahoo Finance 및 RSS 피드는 비공식 엔드포인트로, 속도 제한이나 구조 변경이 예고 없이 발생할 수 있습니다.

> **주의**: 본 시스템은 내부 참고용 초안 작성 도구입니다.
> 중요 의사결정 시에는 반드시 공식 데이터 소스를 병행 확인하시기 바랍니다.

## 🚀 상용 데이터 연결 시 — 완전한 프로덕션 시스템으로

현재 구현은 AI 기반 리포트 생성의 **전체 워크플로우**를 시연합니다.
상용 데이터 피드를 연결하면 위의 모든 한계가 해소되며, 완전한 실전 시스템으로 전환됩니다:

| 업그레이드 | 효과 |
|---|---|
| **Bloomberg B-PIPE / KSDB** | KRW 금리·외환·크레딧 스프레드 실시간 틱 데이터 |
| **Reuters Refinitiv** | NDF, 스왑포인트, Euribor, CDS — 모두 실시간 |
| **KOFIA / KRX 공식 피드** | 장중 채권 수익률, RP 금리, IRS 커브 |
| **FactSet / LSEG** | 기관급 매크로 데이터 및 충분한 역사적 심도 |

상용 데이터 피드를 연결하면, iM AIRGen TD는 **완전한 실시간 AI 리포트 시스템**으로 작동합니다.
장 시작 후 수 분 내에 모닝브리프를 자동 생성·배포하며, 수동 데이터 입력이 전혀 필요 없습니다.

## 보안

- AI 내러티브 API 엔드포인트에 **Bearer 토큰** 인증 적용 (`x-api-token`)
- Anthropic API 키는 **서버 사이드 환경변수**로만 관리 — 브라우저 번들에 노출되지 않음
- ECOS / FRED API 키는 Vercel 환경변수로 관리

## 로컬 개발 환경 설정

```bash
# 1. 클론
git clone https://github.com/yalkongs/iM_AIRGen_TD.git
cd iM_AIRGen_TD

# 2. 패키지 설치
npm install

# 3. 환경변수 설정
# .env.local 파일 생성 후 아래 항목 입력

# 4. 프론트엔드만 실행
npm run dev

# 5. Vercel 함수 포함 로컬 실행
npx vercel dev
```

> 딜러 실무·재무관리·내부통제·CRO/CFO·PB영업·채권 애널리스트·퀀트 등 7개 전문가 에이전트 3차 토론을 통해 도출한 개선 방향입니다.
> 핵심 전제: **"신뢰 없는 속도는 무의미하다"** — 거버넌스 완비 및 고도화

### Phase 1

#### 리포트 내러티브 품질

- **4단 구조 강제화**: 모든 내러티브를 "① 변화 → ② 원인 가설 → ③ 시장 함의 → ④ 전략적 시사점" 구조로 생성
  - 현재 수준: 현상 나열(Bloomberg 화면 복사 수준)
  - 목표 수준: 인과 논리 중심 서술 (KB증권 모닝 노트 수준)
- **리포트 말미 3단 마무리 고정**: 시사점 → 행동 선택지 → 리스크 시나리오
- **전일 대비 판단 변화 이력 표시**: "지난주 대비 스탠스 변경 이유" 명시

#### 데이터 확충 (분석 공백 해소)

| 추가 데이터 | 현재 공백 |
|---|---|
| CD91일, CP91일 | 단기자금시장 스프레드(CD-기준금리) 분석 불가 |
| KTBi (물가연동국채) | BEI 산출·실질금리 분리 불가 |
| 외국인 국채 보유잔고 (ECOS) | 수급 분석의 핵심 지표 누락 |
| 기재부 국채 발행 캘린더 | 발행 수급 효과 미반영 |
| 파생 지표 자동 산출: 10Y-3Y 스프레드, KTB-IRS 스왑 스프레드, BBB-AA 크레딧 스프레드 | 기존 원시 데이터로 즉시 계산 가능, 현재 미제공 |

#### 수신자 맞춤 발송

- **역할별 버전 분기**: 경영진(한 줄 결론 + 리스크 신호등) / 리스크팀(이상 징후 강조) / 영업팀(고객 제안용 요약)
- **긴급 시황 원클릭 프리셋**: 금통위·FOMC·환율 급변 시 5분 이내 즉석 리포트 생성

### Phase 2

#### 리포트 다양화

- **고객군별 3종 템플릿**:
  - 개인 자산가(HNWI): 평어체, 모바일 공유 가능 1페이지 + "지금 달러 예금 들어야 하나" 수준의 즉각적 행동 포인트
  - 중견기업 CFO: 원/달러 방향성, 수출입 결제 타이밍, CP·CD 금리 추이
  - 기관투자자: 스프레드·듀레이션·글로벌 통화정책 연계 퀀트 지표 강조
- **ALM위원회 전용 템플릿**: 월 2회 사이클에 맞춘 별도 포맷, BPV·듀레이션 갭·NII 시나리오 분석 포함
- **Call to Action 삽입**: 리포트 말미에 고객군·시장 상황 기반 상품 연계 제안 문구 자동 생성

#### 분석 고도화

- **스트레스 이상 탐지 복합 지수**: VIX + 한국 CDS5Y + 크레딧 스프레드 + 스왑포인트의 Z-score 동시 이탈 시 경보
- **인터마켓 회귀 모델 파일럿**:
  - `USD/KRW ~ (DXY + UST10Y + KOSPI + 스왑포인트)`
  - `KTB10Y ~ (UST10Y + CPI 서프라이즈 + VIX)`
  - 예측보다 **잔차 1.5σ 이탈 시 괴리 경보** 표시에 집중
- **자기검증 루프 구축**: 전주 전망 vs. 실제 결과 자동 비교 섹션
  - AI가 어떤 판단을 했고 맞았는지를 추적하는 메타 레이어
  - 누적 정확도가 이 시스템의 신뢰 자산 — **금융기관 내 장기 신뢰 획득의 핵심 장치**
- **특정 지표 임계값 초과 시 영업 알림**: 환율 1,450원·금리 특정 레벨 돌파 시 담당 PB에게 자동 알림 + 초안 생성
